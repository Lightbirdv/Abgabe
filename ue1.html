Uebung 1


Aufgabe 1 - 1.1


Musikaufnahme: Abtastfrequenz 44100, stereo, 16 bit depth


Sprachaufnahme: Abtastfrequenz 8000, mono, 16 bit depth


Aufgabe 1 - 1.2

Loesung:

Durch die erhöhte Anzahl an Frames in der Musik muss die Abtastfrequenz erhöht werden damit die tatsaechlichen Schwingungen abgetastet werden und keine ansonsten ausgelassen wird. Da in der Sprachdatei nur eine Person spricht und somit die Anzahl der Frames relativ gering bleibt ist eine kleine Abtastfrequenz von 8000 ausreichend.


Aufgabe 1 - 1.3

Loesung:

Headerausgabe der Musikdatei:
Channels: 2
Frames: 222692
Sample Rate: 44100
Valid Bits: 16
Bytes per sample: 2


Zwei channels bedeutet das der Sound stereo ist und zwei Geräuschestränge gleichzeitig auf 2 channels abgespielt wird. Die hohe Anzahl an Frames ist bei der Musik normal

Headerausgabe der Sprachdatei:
Channels: 1
Frames: 40010
Sample Rate: 8000
Valid Bits: 16
Bytes per sample: 2



Aufgabe 1 - 1.4

Allgemeine Funktion:
Frequency x bit depth x channels = bit rate

Bitrate der Musikdatei:
44100 samples per second x 16 bits per sample x 2 channels = 1411200 bits per second
1411200 bits per second x 5 seconds = 7056000 bit = 882 KB

Bitrate der Sprachdatei:
8000 samples per second x 16 bits per sample x 1 channel = 128000 bits per second
128000 bits per second x 5 seconds = 640000 bit = 80 KB
